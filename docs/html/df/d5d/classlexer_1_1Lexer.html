Detailed Description

This lexer acts as a DFA Note that this DFA is really an NFA, since the checks are those that are defined in ALPHABET_PREDICATES. However, the order of these predicates is indeed a priority, starting from the highest to the lowest and as a result, the Finite-State Automata will not return more than one state, thus making it a deterministic. `
